# Dynamic Group-Based Extended Ping Lock Arbiter

## Overview
This project presents a Verilog-based Dynamic Group-Based Extended Ping Lock Arbiter for shared bus systems. It improves fairness and power efficiency using skip logic and an adaptive grant mechanism.

## Features
- Verilog HDL implementation
- Skip Logic
- Adaptive Grant Mechanism
- Xilinx Vivado Simulation
- Artix-7 FPGA Verification

## Tools Used
- Verilog HDL
- Xilinx Vivado
- Artix-7 FPGA

## Results
- Reduced dynamic power by 63.8%
- Reduced total on-chip power by 58.5%

## Applications
- System-on-Chip (SoC)
- Network-on-Chip (NoC)
- FPGA-based Shared Resource Systems
